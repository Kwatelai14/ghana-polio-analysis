Analyses Performed

📈
OPV3 Coverage Trend
Quarterly coverage rates per district with linear trend lines and statistical significance testing.
🔮
ETS Forecasting
Error-Trend-Seasonality model projecting OPV3 coverage and AFP cases 4 quarters ahead with 80% and 95% confidence intervals.
🌀
STL Decomposition
Seasonal-Trend decomposition separating the trend, seasonal, and remainder components of coverage series.
🚨
AFP Surveillance
Acute Flaccid Paralysis case counts with 3-period moving average for smoothed surveillance signal detection.
💊
Dropout Analysis
OPV1→OPV3 dropout rates measuring children who started but did not complete the vaccine series.
🔍
Zero-Dose Tracking
Children who received no OPV dose — a WHO/UNICEF equity priority indicator for outreach targeting.

Quick Start

Run in R

# Clone and run
git clone https://github.com/
  YOUR_USERNAME/ghana-polio-analysis

setwd("ghana-polio-analysis")
source("R/run_all.R")

# Outputs saved to outputs/
# dashboard_full.png + 6 charts

Connect to DHIS2 API

# In R/01_load_clean.R
USE_API <- TRUE

DHIS2_BASE_URL <- "https://your-instance
  .dhis2.org"
DHIS2_USERNAME <- "your_username"
DHIS2_PASSWORD <- "your_password"

# Then run:
source("R/run_all.R")

Deploy GitHub Pages

git add index.html
git commit -m "Add GitHub Pages"
git push

# In repo Settings:
# Pages → Deploy from branch
# Branch: main / root
# → Save
